# TurfFindr

## Introduction

TurfFindr is a place where users can rent out their Turf, and book others' Turf. 

## Getting Started

First, add the credentials for the MySQL database in .env. An example .evn is provided.

Second, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
